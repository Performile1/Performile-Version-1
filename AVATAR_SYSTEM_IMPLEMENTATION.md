## üé® COLOR GENERATION ALGORITHM

### **How It Works:**

```typescript
const stringToColor = (str: string): string => {
  // 1. Generate hash from string
  let hash = 0;
  for (let i = 0; i < str.length; i++) {
    hash = str.charCodeAt(i) + ((hash << 5) - hash);
  }
  
  // 2. Convert hash to HSL color
  const hue = Math.abs(hash % 360);           // 0-360 degrees
  const saturation = 65 + (Math.abs(hash) % 20); // 65-85%
  const lightness = 45 + (Math.abs(hash) % 15);  // 45-60%
  
  return `hsl(${hue}, ${saturation}%, ${lightness}%)`;
};
```

### **Why HSL?**
- ‚úÖ Consistent saturation = vibrant colors
- ‚úÖ Consistent lightness = readable text
- ‚úÖ Full hue range = diverse colors
- ‚úÖ Same name always gets same color
- ‚úÖ White text always readable on background

## üîÑ FALLBACK BEHAVIOR

### **Image Loading Flow:**
```
1. Component renders with imageUrl
   ‚Üì
2. Image starts loading
   ‚Üì
3a. Image loads successfully
    ‚Üí Display image
    
3b. Image fails to load (404, network error)
    ‚Üí onError triggered
    ‚Üí Hide image element
    ‚Üí Show colored circle with initials
```

### **No Image URL:**
```
1. Component renders without imageUrl
   ‚Üì
2. Immediately show colored circle with initials
```
