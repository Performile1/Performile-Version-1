## üé® TRUSTSCORE INTEGRATION

### What is TrustScore?

**TrustScore** is Performile's proprietary courier rating system that combines:
- Customer reviews (25%)
- On-time delivery rate (20%)
- Completion rate (20%)
- Response time (10%)
- Customer satisfaction (10%)
- Issue resolution (5%)
- Delivery attempts (5%)
- Last-mile performance (5%)

**Score Range:** 0-100  
**Grades:** A+, A, A-, B+, B, B-, C+, C, D

### Checkout Integration (NEW):

**Before (Old):**
- Courier list with price and delivery time
- No ratings visible
- No trust indicators
- Generic selection

**After (V3.5):**
- TrustScore-ranked list (highest first)
- Star ratings and review counts prominent
- Performance metrics visible (on-time, completion)
- Badges highlight features (top rated, fastest, best value)
- Link to detailed TrustScore page
- Informed, data-driven selection

**Expected Impact:**
- +10-15% checkout conversion
- Higher customer satisfaction
- Better courier selection
- Competitive advantage
